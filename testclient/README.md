# Riddle-Me Test Client

This directory contains a **simple client environment used to test and experiment with backend features**.

The test client is **not intended to be a production client**. Instead, it acts as a lightweight space where developers can quickly create small HTML pages to test server functionality.

## Purpose

The client allows developers to:

- Connect to the backend server
- Test real-time features
- Experiment with new backend functionality
- Quickly build small pages that exercise server behavior

For example, developers may add small pages that test:

- voice communication
- signaling behavior
- new backend APIs
- experimental features

## Notes

The test client should grow alongside the backend. As new backend features are introduced, simple pages can be added here to verify functionality and help prevent regressions, **ideally built in a way that mimics user flow or a dedicated QA suite**.

The goal is to provide a **fast and flexible testing environment** so developers can focus on building backend features without needing a full application frontend.
